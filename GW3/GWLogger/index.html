Gateway: Sessions:

Welcome to the EPICS Gateway Log Analyzer

How to use it:

  1. Select the gateway you want to check on the top left side. The gateway sessions are then loaded and displayed on the left bar, as well as the statistics are displayed on the time range widget.
  2. You can click the time range widget to check out more details on that time.

Data retention:

Data is kept for a max. time of 10 days.
