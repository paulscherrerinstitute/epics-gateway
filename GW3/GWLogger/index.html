Gateway: Sessions:

Welcome to the EPICS Gateway Log Analyzer

How to use it:

  1. Select the gateway you want to check on the top left side. The gateway sessions are then loaded and displayed on the left bar, as well as the statistics are displayed on the time range widget.
  2. You can click (and drag) the time range widget to check out more details on that time.
  3. The time range shows 3 values: top light green bars are the number of event logs, bottom dark green bars are the number of searches, red lines are the number of errors. The graphs uses 3 different scales.
  4. Logs uses UTC time (avoiding issue with day light saving time).
  5. The current UTC time is displayed on the bottom left corner of the screen.
  6. Each bar of the time range is a 10 min range.

Data retention:

Data is kept for a max. time of 10 days.
